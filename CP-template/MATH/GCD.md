### 一、模板类别

​	数学：最大公约数（ `GCD` ），最小公倍数（ `LCM` ）。

### 二、模板功能

#### 1.求最大公约数

1. 数据类型

   模板参数 `typename _Elem` ，可以为 `uint32_t` 或者 `uint64_t` ，表示元素类型。

   输入参数 `_Elem a` ，表示第一个数。

   输入参数 `_Elem b` ，表示第一个数。

2. 时间复杂度

   $O(\log n)$ 。

3. 备注

   本模板基于 `stein` 算法，比网上的欧几里得算法更优。

#### 2.求最小公倍数

1. 数据类型

   模板参数 `typename _Elem` ，可以为 `uint32_t` 或者 `uint64_t` ，表示元素类型。

   输入参数 `_Elem a` ，表示第一个数。

   输入参数 `_Elem b` ，表示第一个数。

2. 时间复杂度

   $O(\log n)$ 。

3. 备注

   本模板基于 `stein` 算法，比网上的欧几里得算法更优。

   **注意：** 当结果超出 `64` 位无符号整数范围时，无法返回正确答案。

### 三、模板示例

```c++
#include "MATH/GCD.h"
#include "IO/FastIO.h"

int main(){
    cout<<"gcd of (24,102) = "<<OY::gcd(24,102)<<endl;
    cout<<"lcm of (24,102) = "<<OY::lcm(24,102)<<endl;
}
```

```
#输出如下
gcd of (24,102) = 6
lcm of (24,102) = 408

```
